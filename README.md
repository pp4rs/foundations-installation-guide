## Building the Site

We use [MkDocs]() to build the the course websites for this course.

* Install MkDocs with the following command:
```{.bash, id:"j29ie3c7"}
pip install mkdocs
```

* Look at a copy of the site served locally on your machine:
```{.bash, id:"j29ie3c7"}
mkdocs serve
```
* Post the site to github pages using:
```{.bash, id:"j29ie3c7"}
mkdocs gh-deploy
```
